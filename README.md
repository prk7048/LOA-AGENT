# 🛡️ LOA AGENT v2: Lost Ark Integrated Management Service

> **"Data-Driven Growth, AI-Powered Strategy"** > 로스트아크 플레이어의 성장을 돕는 에이전트

## 📖 프로젝트 소개 (Introduction)
**LOA AGENT v2**는 기존 학습용 프로젝트(v1)의 기술 부채를 해결하고, **실제 상용 서비스 수준의 아키텍처**를 목표로 새롭게 설계된 프로젝트입니다. 단순한 기능 구현을 넘어, 확장성 있는 구조(Scalable Architecture)와 배포 자동화(CI/CD)를 포함한 **엔지니어링 역량 강화**에 중점을 두었습니다.

## 개발 목표 (Engineering Goals)
- **Service-Oriented Architecture:** 단순 로컬 스크립트가 아닌, 다수 유저가 사용 가능한 웹 서비스 구조 설계
- **Clean Code & Modularization:** 유지보수와 확장이 용이한 MVC 패턴 및 모듈화 적용
- **DevOps Practice:** Docker 컨테이너화 및 AWS 클라우드 배포 경험

## 🛠️ 기술 스택 (Tech Stack)
- **Frontend/Backend:** Python 3.11, Streamlit
- **Database:** PostgreSQL 16 (Relational DB)
- **Data Engineering:** LostArk Official API, Custom Crawler
- **AI/LLM:** Context-aware Advisor (RAG 기반 예정)
- **Deployment:** Docker, AWS EC2

## ✨ 주요 기능 (Key Features)
1.  **Integrated Dashboard:** 숙제 체크, 골드 수급 현황, 원정대 정보를 한눈에 파악
2.  **Market Intelligence:** 경매장 시세 추적 및 효율적인 아이템 구매 가이드
3.  **Growth Diary:** 아이템 레벨/전투력 성장 추이 시각화 및 회고(Diary) 기능
4.  **AI Commander:** 게임 패치 노트, 시장 동향, 내 스펙을 종합 분석하여 성장 방향성 제안
